# Решение проблемы с обработкой роутера

## Проблема
Коллектор пытался подключиться к роутеру `rtr` (192.168.1.1), хотя он должен быть исключен из обработки.

## Что было исправлено

### 1. Улучшенная фильтрация хостов
- Добавлено исключение по имени хоста (`rtr`, `router`, `gateway`)
- Добавлена проверка IP адресов в SSH конфигурации
- Автоматическое исключение локальных сетей:
  - 192.168.x.x
  - 10.x.x.x  
  - 172.16-31.x.x

### 2. Обновленные значения по умолчанию
```python
# Старая версия
exclude_hosts = ["192.168.1.1"]

# Новая версия  
exclude_hosts = ["192.168.1.1", "rtr"]
```

### 3. Интеллектуальная проверка
Функция `_should_include_host()` теперь проверяет:
- Имя хоста в SSH config
- IP адрес (hostname) в SSH config
- Паттерны локальных сетей
- Wildcards в именах хостов

## Автоматический анализ

### Новая функциональность
После успешного сбора логов система автоматически предлагает:
1. Объединить логи с нескольких серверов
2. Отсортировать по времени
3. Запустить графический анализ

```bash

```
